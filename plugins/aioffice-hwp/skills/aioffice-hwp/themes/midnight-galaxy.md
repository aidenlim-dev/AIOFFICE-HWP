# 미드나잇 — 드라마틱 퍼플

Anthropic theme-factory의 **Midnight Galaxy**을 AIOFFICE-HWP(HWPX/HWP)용으로 변환한 테마.

- **제목 색**: `#2B1E3E`
- **강조 색(accent)**: `#4A4E8F`
- **제목 글꼴**: Pretendard Black
- **본문 글꼴**: Pretendard
- **용도**: 엔터·게이밍·나이트라이프·럭셔리

> 변환 노트: 색 팔레트는 원본 유지(`#2b1e3e` `#4a4e8f` `#a490c2` `#e6e6fa`), 글꼴만 한컴
> 웹·앱에서 렌더되는 A세트로 매핑(원본 DejaVu/FreeSans 계열은 한컴에서 대체됨).
